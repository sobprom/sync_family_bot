# 🛒 SyncFamily Shopping List Bot

Реактивный Telegram-бот на стеке **Quarkus + Mutiny**, предназначенный для ведения общего списка покупок внутри семьи с мгновенной синхронизацией интерфейса.

## 🌟 Ключевые особенности

*   **Family Sync**: Любое изменение списка (добавление, покупка, удаление) приводит к автоматическому обновлению сообщений у всех членов семьи.
*   **Clean Chat**: Бот удаляет старые сообщения со списком (`DeleteMessage`) перед отправкой новых, чтобы актуальный список всегда был последним в чате.
*   **Интеллектуальный интерфейс**:
    *   **View Mode**: Быстрая отметка купленных товаров (✅).
    *   **Edit Mode**: Режим управления с возможностью переименования и точечного удаления (❌) через промежуточное меню подтверждения.
*   **Реактивность**: Полностью неблокирующая обработка запросов на базе `Uni` и воркер-пулов инфраструктуры Quarkus.

## 🛠 Технологии

*   **Framework**: Quarkus
*   **Reactive**: SmallRye Mutiny
*   **Database**: PostgreSQL (через PostgresDb async wrapper)
*   **Telegram SDK**: TelegramBots (Long Polling)
*   **Utilities**: Lombok, Apache Commons Lang3
